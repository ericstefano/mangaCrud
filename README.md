## Install

```sh
npm install
```

## Usage

Run:

```sh
npm run prod
```

On another terminal, run:

```sh
npm run json-server
```

Finally, open another terminal and run:

```sh
npm run live-server
```

This should open [localhost:1234](http:localhost:1234) on your browser.
